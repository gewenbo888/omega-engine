## Honest framing

OMEGA is not literal generative AI. It is a structured conceptual atlas — a thinking tool. The "self-expanding" property is implemented via genuinely interactive elements (universe selector, branching tree, counterfactual generator, depth slider) that respond to user input. Each layer presents established theory + open questions; speculative claims clearly marked.
